### 概述
主要包括两个页面
- 首页，显示肺炎的分布
  - 可以查询历史上每一天的数据
  - 可以选择显示省份还是城市的数据
  - 可以选择数据类型：确诊、治愈还是死亡
![home page](/doc/home.jpg)
- 趋势，显示指定区域的相关数据的历史走势，还在开发中
